Seja Bem Vindo ao PX-DOS

Primeiramente, seja bem vindo ao PX-DOS. É um prazer recebe-lo. Para começar, uma rápida apresentação. O PX-DOS é um sistema operacional desenvolvido para arquiteturas x86, criado para ser rápido, estável e fácil de ser manuseado.

O que ele pode fazer? O que ele é?

O PX-DOS foi desenvolvido para proporcionar a você a possibilidade de utilizar alguns aplicativos antigos, corretamente certificados. O PX-DOS pode ser utilizado desde computadores antigos a computadores mais modernos, oferecendo a mesma fluidez e estabilidade. Ele é um sistema operacional criado para computadores baseados na arquitetura x86. É um sistema leve, fluido, prático e fácil de ser utilizado.

O Sistema

Velocidade

O PX-DOS te entrega todo o controle de sua máquina. Desenvolvido para ser rápido, aproveita todos os recursos de sua máquina. A maioria de seus aplicativos é desenvolvida em Assembly, uma linguagem de programação de baixo nível, que visa a fluidez e economia de recursos. O PX-DOS é o parceiro ideal de seu computador. Foi feito para ele.

Aplicativos

O sistema traz consigo uma coleção de aplicativos nativos para seu conforto e divertimento. Temos aplicativos para tudo: desde uma simples calculadora até um piano virtual. Os aplicativos foram desenvolvidos seguindo as filosofias do sistema: fluidez e simplicidade. Todos os aplicativos nativos se integram perfeitamente ao sistema, e se completam. Existem também aplicativos de terceiros, que podem ser instalados em seu computador, para aproveitar ainda mais o sistema e seu computador.

Tamanho

Todo o sistema foi pensado para ser leve. Leve ao extremo. Todo o sistema cabe em apenas um disquete! Realmente, disquete é coisa do passado... Todo o sistema ocupa apenas 1 Megabyte em seu computador. Claro! Quanto mais espaço para o usuário melhor.
